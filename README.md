# DOPPIO DARK THEME

## About the theme

- minimal distractions
- Yellowish colors for greater eye comfort in low-light environments.
- low saturation + low contrast colors

## How to install

- download the "[**doppio-dark-0.0.1.vsix**](https://github.com/mat-pellizzari/doppio-dark-theme/raw/refs/heads/master/doppio-dark-0.0.1.vsix)" file
- run code --install-extension doppio-dark-0.0.1.vsix **or** copy it into the "<user home>/.vscode/extensions"
